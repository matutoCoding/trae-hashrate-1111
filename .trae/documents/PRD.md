## 1. 产品概述
公司行政公章用印管理后台系统，解决企业公章使用审批、效期管理及用印登记的全流程数字化管理。

- 核心目标：实现用印申请多级审批流转、印章效期全生命周期管理、用印过程可追溯存证
- 目标用户：公司行政人员、部门负责人、分管领导、印章管理员

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 普通员工 | 账号登录 | 提交用印申请、查看申请状态 |
| 部门负责人 | 账号登录 | 审批本部门用印申请、驳回退回 |
| 分管领导 | 账号登录 | 审批分管部门用印申请、驳回退回 |
| 印章管理员 | 账号登录 | 印章管理、用印登记、拍照存证 |
| 系统管理员 | 账号登录 | 全部权限、用户管理 |

### 2.2 功能模块

1. **用印申请模块**：申请单创建、申请列表、申请详情、编辑草稿
2. **多级审批模块**：待我审批、审批操作、审批轨迹、驳回退回上一步
3. **印章效期模块**：印章入库、效期管理、临期预警、过期锁定、先进先出
4. **用印登记模块**：用印登记、拍照存证、用印记录查询

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|-----------|
| 仪表盘 | 数据概览 | 待审批数量、临期印章预警、本月用印统计、用印类型分布 |
| 用印申请 | 申请列表 | 按状态筛选、搜索、新建申请、查看详情 |
| 用印申请 | 新建/编辑申请 | 选择印章类型、用印事由、用印文件、用印数量、紧急程度 |
| 多级审批 | 待审批列表 | 展示待本人审批的申请单，支持筛选搜索 |
| 多级审批 | 审批详情 | 展示申请详情、审批轨迹、同意/驳回操作 |
| 多级审批 | 审批轨迹 | 时间轴展示各节点审批人、审批意见、审批时间 |
| 印章效期 | 印章列表 | 展示所有印章批次、状态、效期、启用状态 |
| 印章效期 | 印章入库 | 新增印章批次、录入编号、效期、保管人 |
| 印章效期 | 临期预警 | 展示30天内到期的印章，高亮预警 |
| 用印登记 | 登记列表 | 已用印记录查询、筛选 |
| 用印登记 | 用印登记 | 关联审批通过的申请、拍照上传、登记用印人信息 |

## 3. 核心流程

### 3.1 用印审批流程

员工提交用印申请 → 部门负责人审批（同意/驳回） → 分管领导审批（同意/驳回） → 印章管理员用印登记

任一审批节点可驳回退回上一步，申请人修改后重新提交。

```mermaid
flowchart LR
    A["员工提交申请"] --> B["部门负责人审批"]
    B -->|同意| C["分管领导审批"]
    B -->|驳回| A
    C -->|同意| D["印章管理员登记用印"]
    C -->|驳回| B
```

### 3.2 印章使用流程

```mermaid
flowchart LR
    A["印章入库"] --> B["启用（先进先出")
    B --> C["正常使用"]
    C --> D["临期预警（30天）
    D --> E["过期锁定"]
    E --> F["停用/销毁"]
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝 #1e3a5f（稳重、专业）
- 辅助色：红色 #dc2626（预警、驳回）、绿色 #16a34a（通过、正常）、橙色 #f97316（临期、警告）
- 背景色：浅灰 #f8fafc
- 按钮风格：圆角6px，悬停阴影效果
- 字体：Noto Sans SC（中文）
- 布局风格：左侧导航 + 顶部栏 + 内容卡片式布局
- 图标风格：Lucide 图标库

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|-------|
| 仪表盘 | 数据概览 | 统计卡片、预警卡片、环形图、柱状图 |
| 用印申请 | 申请列表 | 数据表格、状态标签、筛选器 |
| 多级审批 | 审批详情 | 信息卡片、时间轴、操作按钮组 |
| 印章效期 | 印章列表 | 状态徽章、效期进度条、预警高亮 |
| 用印登记 | 登记页面 | 图片上传、表单、记录列表 |

### 4.3 响应式设计
- 桌面端优先设计（≥1280px）
- 平板端自适应（≥768px）
- 移动端支持（≥375px）
